# My Portfolio
A personal portfolio website created using HTML, CSS, and JavaScript.

## Structure
- `index.html`: Main HTML file.
- `style.css`: Styling for the site.
- `script.js`: JavaScript for functionality.
- `assets/images`: Folder for images.

## How to Edit
- To update the skills, edit the `<div class="skills-container">` in `index.html`.
- To add projects, edit the `<section id="projects">` in `index.html` and add images to `assets/images/`.

## Deployment
- Push changes to GitHub, and the site will update at https://yourusername.github.io/portfolio.
